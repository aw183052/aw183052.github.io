Title: Impact of Earthquakes from 1970 to 2020¶

Team members: Alan Chen and Samuel Chen¶

Motivation:¶

Within the past decade, Earth has experienced more destructive severe natural disasters. Earthquakes and other natural disasters have increased in frequency over the past decade. Earthquakes are also known as a natural disaster that can cause additional destruction by triggering other disasters like tsunamis and bolcano eruptions. We will investigate earthquakes records from the past in this project.

This project aims to analyze trends and the impacts related to earthquakes between the years 1970 to 2020. We chose to analyze this time interval since research had found that climate change had an effect on weather patterns. This time interval is recent enough to get a general idea of Earths current seismic activity and old enough to create a regression model for modern seismic activity. In this project we will investigate:
        The relationships between frequency and locations of earthquakes
        Magnitude and damage (in Millions of $)
        Magnitude and injuries
        Magnitude and deaths
        Which countries have the most earthquakes
        The number of earthquakes per year.

As climate change continues to be a pressing problem for the future, data analysis of this project will contribute to predictions of Earthquake frequency, severity, and locations so we can identify countries which are most vulnerable to earthquakes and project how earthquakes could look like in the future.

Data Set Selection:¶

The dataset that we selected to analyze for this project is the Earthquake data set from the National Geophysical Data Center (NGDC) which is a subdivision of the National Oceanic and Atmospheric Administration. We chose this data set since it is a credible source since it is regulated by the US government. This data set contained records dating back to 2586 BC to the present. We recognized that the present seismic landscape to be different since then and selected the time interval between 1970 till the present.

Tidying Data:¶

Raw Data Set¶

Initially the data that we used had a total of 39 columns as shown above. Many of the columns have NaN as a value for an earthquake entry. We decided to remove those columns because all those missing values would not be helpful with answering our questions about earthquakes. Furthermore, there are some columns that contain information that are not required for us to complete our investigation. For example, the longitude and latitudes of these earthquakes are not needed to answer our investigation questions listed above.

When tidying the data we found that some of the columns were repeated columns. Examples include Deaths and Total Deaths columns, Missing and Total Missing, Injuries and Total Injuries, and Damage and Total Damage. To resolve this we decided to just use one of these columns to represent that parameter. Unfortunately, a significant amount of these values were still missing. Instead we used the Description columns to measure those parameters since there was a much higher turnout in the raw data set.

To clean up the columns for the datetime, we combined the "Year", Mo", "Dy", "Hr", "Mn", "Sec" columns.

The remaining columns, Mag and Location Name, from the raw data set were left untouched. One additional column that we made was the country column. This was created using the Location Name column. Since that column had the format Country, City we created that column by getting the first portion of the Location Name column.

We tidied the data set based on the columns that we need to answer the questions in our investigation. The columns that we have are:

1) Datetime - Represents the date and the time an earthquake occured
2) Location Name - Lists the City and Country the earthquake occurred
3) Country - List the country the earthquake occurred
4) Mag - Magnitude of the earthquake
5) Damage Description - Scales 0 - 4 to measure the approximate amount of damage done in millions
        0 - indicates No damage done
        1 indicates Limited damage(approximately less than \$1 million)<br/> &nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2 indicates moderate damage (approximately \$1-\$5 million)<br/> &nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3 indicates severe damage (approximately \$5-\$25 million)<br/> &nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;4 indicates Extreme damage(approximately \$25+ million)

6) Injury Description - Scales 0 - 4 to measure the approximate amount of injuries reported
        0 indicates None
        1 indicates Few (1-50 injuries)
        2 indicates Some (51-100 injuries)
        3 indicates Many (101 - 1000 injuries)
        4 indicates Very Many (over 1000 injuries)

7) Deaths Description- The number of deaths listed
0 indicates None
        1 indicates Few (1-50 Deaths)
        2 indicates Some (51-100 Deaths)
        3 indicates Many (101 - 1000 Deaths)
        4 indicates Very Many (over 1000 Deaths)

# function that runs linear regression & prints plot with statistics
def lin_reg(x, y):
  # a random_state is set for stablized accuracy score; is optional
  x_train, x_test, y_train, y_test = train_test_split(x, y, random_state=10)
  
  # create the linear regression model
  lin_model = linear_model.LinearRegression()

  # fit the training data
  lin_model.fit(x_train, y_train)

  # get the predictions based on the test set after fitting model
  y_prediction = lin_model.predict(x_test)

  # figure out the r^2 score, the slope, and intercept of the linear regression created by the model
  print(f'Accuracy of trained model matching the ground truth (R^2): {lin_model.score(x_test, y_test)}')
  print(f"Model's coefficient is: {lin_model.coef_[0]}")
  print(f"Model's intercept is: {lin_model.intercept_}")

  # return the test set input and the predicted values
  return x_test, y_prediction

Average Damage, Injuries and Deaths per Year¶

These regression lines look at 3 different parameters: amount of damage(Red), injuries (Blue), and deaths (Green) between the years 1970 and 2020. The red data points and regression line represents the damage description. The calculated model is Y = -0.0058536X + 13.79629366 and the r2 value is 0.12167200106992637. The negative slope indicates that the amount of damage being done is decreasing over time. However it is important to note that the coefficient of determination is very weak meaning that this is not the most accurate model.

Next, there is the blue data points and regression line which represents the amount of damage. The calculated regression line is Y = -0.00997791 + 21.82645275 and the r2 is 0.1746699066819657. The negative slope indicates that the number of injuries have been decreasing since 1970. However, since the coefficient is close to 0, this model may not be accurate for this dataset. This error could be due to a faulty dataset.

Lastly, the green data points and regression line represent the deaths that occured over the years. The calculated regression line is Y = -0.0082716X + 17.90497595 and the correlation coefficient r2 is 0.25804961207832433. This is shown to have a negative correlation, meaning that the number of deaths have been decreasing. However, the correlation coefficient is not high meaning that this model is not very accurate. This could be due to faulty dataset.

From these three parameters, they all have a negative correlation however do not have a convincing correlation coefficient. This could potentially be due to the dataset not having a large enough sample to have an accurate prediction, or due to a change in the data collection methods over the years. If these models all have correlation coefficients close to one, it would mean that human response to earthquake intervention has improved. Examples of this could be having more explicit protocols for earthquake responses to increase safety, improved structural engineering which makes buildings more resistant to earthquakes, improved medical procedures to treat injured victims or more.

Average Magnitude Over Time (Years)¶

This is the graph of the average magnitude per year. The data points were calculated by adding all the magnitudes of all the earthquakes in each year and dividing it by the total number of earthquakes in that year. The linear regression model is Y = -0.0073966X + 0.25804961207832433 and the correlation coefficient r2 is 0.347173994598746. This indicates a small negative correlation with a mediocre correlation coefficient. Since 1970, the severity of earthquakes have been decreasing gradually.

Connecting to the 3 linear regrssion graphs from above, we can tell that the injuries and death levels are decreasing at faster rates compared to that of magnitude. This implies a growth in solutions for earthquakes, yet damage level is decreasing at a lower rate than that of magnitude. This could potentially be due to the fact that we are constructing more and more facilities and building, which tend to cost more. Since damage level is measured by the rough amount of money lost, it may be reasonable to see a slower decline of damage levels in the future as well.
